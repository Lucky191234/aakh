# Aakh

**Live Demo:** https://lucky191234.github.io/aakh/

Aakh is a one-stop dashboard built for enthusiasts, students, engineers,
and hobbyists who want to stay updated without switching between
multiple platforms.

The dashboard brings together the latest technology news, upcoming
hackathons, and relevant GitHub repositories in a clean,
distraction-free interface. Content refreshes automatically every day,
while important articles and repositories can be saved to persist across
updates.

To make staying informed easier during travel or multitasking, Aakh also
includes a brief audio summary feature that reads key content aloud.

## Features

-   Latest technology news
-   Upcoming hackathons
-   Curated GitHub repositories
-   Save news and repositories for later
-   Daily automatic content refresh
-   Audio summaries for hands-free listening
